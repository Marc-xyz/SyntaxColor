	<h1 class="firstHeading" id="firstHeading" lang="en">Bittering agent</h1>
	
	<div class="mw-body-content" id="bodyContent">
		<div class="noprint" id="siteSub">From Wikipedia, the free encyclopedia</div>
		<div id="contentSub"></div>
		
		
		
		<div id="jump-to-nav"></div>
		<a class="mw-jump-link" href="#mw-head">Jump to navigation</a>
		<a class="mw-jump-link" href="#p-search">Jump to search</a>
		<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><div class="hatnote navigation-not-searchable" role="note">This article is about the use as a food additive. For use as an aversive agent, see <a href="/wiki/Bitterant" title="Bitterant">Bitterant</a>.</div>
<p>A <b>bittering agent</b> is a <a class="mw-redirect" href="/wiki/Flavoring_agent" title="Flavoring agent">flavoring agent</a> added to a food or beverage to impart a <a class="mw-redirect" href="/wiki/Bitter_(taste)" title="Bitter (taste)">bitter taste</a>, possibly in addition to other effects. While many substances are bitter to a greater or lesser degree, a few substances are used specifically for their bitterness, especially to balance other flavors, such as sweetness. Notable beverage examples include <a href="/wiki/Caffeine" title="Caffeine">caffeine</a>, found naturally in <a href="/wiki/Tea" title="Tea">tea</a> and <a href="/wiki/Coffee" title="Coffee">coffee</a> and added to many <a class="mw-redirect" href="/wiki/Soft_drinks" title="Soft drinks">soft drinks</a>, <a href="/wiki/Hops" title="Hops">hops</a> in <a href="/wiki/Beer" title="Beer">beer</a>, and <a href="/wiki/Quinine" title="Quinine">quinine</a> in <a href="/wiki/Tonic_water" title="Tonic water">tonic water</a>.
</p><p>Food examples include <a class="mw-redirect" href="/wiki/Bitter_melon" title="Bitter melon">bitter melon</a>, which may be mixed into a <a class="mw-redirect" href="/wiki/Stir_fry" title="Stir fry">stir fry</a> or soup for its bitter flavor.
</p><p>Bittering agents are also used as additives in detergents and other household cleaning products to prevent accidental poisoning.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> However, the efficacy of using bittering agents for this purpose is not conclusive. 
</p>

<h2><span class="mw-headline" id="Beer">Beer</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bittering_agent&amp;action=edit&amp;section=1" title="Edit section: Beer">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Prior to the introduction of hops, many other bitter herbs and flowers were used as bittering agents in beer, in a mixture called <a href="/wiki/Gruit" title="Gruit">gruit</a>, which could include <a class="mw-redirect" href="/wiki/Dandelion" title="Dandelion">dandelion</a>, <a class="mw-redirect" href="/wiki/Burdock" title="Burdock">burdock root</a>, <a href="/wiki/Calendula" title="Calendula">marigold</a>, <a class="mw-redirect mw-disambig" href="/wiki/Horehound_(disambiguation)" title="Horehound (disambiguation)">horehound</a> (the German name for horehound means "mountain hops"), <a class="mw-redirect" href="/wiki/Ground_ivy" title="Ground ivy">ground ivy</a>, and <a href="/wiki/Calluna" title="Calluna">heather</a>.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> Also <a class="mw-redirect" href="/wiki/Bog_myrtle" title="Bog myrtle">bog myrtle</a>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>
</p><p>More recently, some <a href="/wiki/Beer_in_China" title="Beer in China">Chinese</a> and Okinawan beer uses <a class="mw-redirect" href="/wiki/Bitter_melon" title="Bitter melon">bitter melon</a> as a bittering agent.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> 
</p>
<h2><span class="mw-headline" id="Other_substances">Other substances</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bittering_agent&amp;action=edit&amp;section=2" title="Edit section: Other substances">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Various other substances are used, including:
</p>
<ul><li><a href="/wiki/Aloin" title="Aloin">Aloin</a></li>
<li><a class="mw-redirect" href="/wiki/Gesho" title="Gesho">Gesho</a>, used in <a href="/wiki/Tej" title="Tej">Tej</a>, Ethiopian honey wine</li></ul>
<h2><span class="mw-headline" id="Other_uses">Other uses</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bittering_agent&amp;action=edit&amp;section=3" title="Edit section: Other uses">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Other prominent uses of bittering agents include:
</p>
