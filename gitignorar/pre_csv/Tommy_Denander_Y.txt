<p><b>Tommy Denander</b> (born March 10, 1968 in <a href="/wiki/Stockholm" title="Stockholm">Stockholm</a>, <a href="/wiki/Sweden" title="Sweden">Sweden</a>) is a <a href="/wiki/Guitarist" title="Guitarist">guitarist</a>, <a class="mw-redirect" href="/wiki/Song_writer" title="Song writer">song writer</a> and <a class="mw-redirect" href="/wiki/Music_producer" title="Music producer">producer</a>. He is mostly famous for his role in the <a href="/wiki/Album-oriented_rock" title="Album-oriented rock">AOR</a> project Radioactive.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>
</p><p>He is the founder of Legends of Rock, a project that featured rock singers including <a href="/wiki/Bobby_Kimball" title="Bobby Kimball">Bobby Kimball</a>, <a href="/wiki/Jimi_Jamison" title="Jimi Jamison">Jimi Jamison</a>, <a href="/wiki/Joe_Lynn_Turner" title="Joe Lynn Turner">Joe Lynn Turner</a>, <a href="/wiki/Tony_Martin_(British_singer)" title="Tony Martin (British singer)">Tony Martin</a>, <a href="/wiki/Graham_Bonnet" title="Graham Bonnet">Graham Bonnet</a>, <a href="/wiki/Mickey_Thomas_(singer)" title="Mickey Thomas (singer)">Mickey Thomas</a>, <a class="mw-redirect" href="/wiki/Fergie_Frederiksen" title="Fergie Frederiksen">Fergie Frederiksen</a>, <a href="/wiki/Steve_Augeri" title="Steve Augeri">Steve Augeri</a>, and <a href="/wiki/Eric_Martin_(musician)" title="Eric Martin (musician)">Eric Martin</a>.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>
</p><p>His signature VGS guitar features both the <a href="/wiki/Evertune" title="Evertune">Evertune</a> bridge and True Temperament frets.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>
</p>
