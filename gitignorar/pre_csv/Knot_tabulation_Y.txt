	<h1 class="firstHeading" id="firstHeading" lang="en">Knot tabulation</h1>
	
	<div class="mw-body-content" id="bodyContent">
		<div class="noprint" id="siteSub">From Wikipedia, the free encyclopedia</div>

		<a class="mw-jump-link" href="#p-search">Jump to search</a>
		<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><div class="thumb tright"><div class="thumbinner" style="width:302px;"><a class="image" href="/wiki/File:Knot_table.svg"><img alt="" class="thumbimage" data-file-height="350" data-file-width="470" decoding="async" height="223" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/12/Knot_table.svg/300px-Knot_table.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/12/Knot_table.svg/450px-Knot_table.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/12/Knot_table.svg/600px-Knot_table.svg.png 2x" width="300"/></a>  <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/File:Knot_table.svg" title="Enlarge"></a></div>A small table of all <a class="mw-redirect" href="/wiki/Prime_knots" title="Prime knots">prime knots</a> (excluding <a href="/wiki/Chiral_knot" title="Chiral knot">mirror images</a>) with 7 <a href="/wiki/Crossing_number_(knot_theory)" title="Crossing number (knot theory)">crossings</a> or fewer.</div></div></div>
<p>Ever since <a class="mw-redirect" href="/wiki/Sir_William_Thomson" title="Sir William Thomson">Sir William Thomson</a>'s <a href="/wiki/History_of_knot_theory" title="History of knot theory">vortex theory</a>, mathematicians have tried to classify and tabulate all possible <a href="/wiki/Knot_(mathematics)" title="Knot (mathematics)">knots</a>. As of May 2008, all <a href="/wiki/Prime_knot" title="Prime knot">prime knots</a> up to 16 <a href="/wiki/Crossing_number_(knot_theory)" title="Crossing number (knot theory)">crossings</a> have been tabulated.<sup class="reference" id="cite_ref-htw_1-0"><a href="#cite_note-htw-1">[1]</a></sup>
</p>

<h2><span class="mw-headline" id="Beginnings">Beginnings</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Knot_tabulation&amp;action=edit&amp;section=1" title="Edit section: Beginnings">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable" role="note">Main article: <a href="/wiki/History_of_knot_theory" title="History of knot theory">History of knot theory</a></div>
<p>In an attempt to make a <a class="mw-redirect" href="/wiki/Periodic_table_of_the_elements" title="Periodic table of the elements">periodic table of the elements</a>, <a class="mw-redirect" href="/wiki/P._G._Tait" title="P. G. Tait">P. G. Tait</a>, <a class="mw-redirect" href="/wiki/C._N._Little" title="C. N. Little">C. N. Little</a> and others started counting all possible knots.<sup class="reference" id="cite_ref-Hoste_2-0"><a href="#cite_note-Hoste-2">[2]</a></sup>
</p>
<h2><span class="mw-headline" id="Perko_pair">Perko pair</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Knot_tabulation&amp;action=edit&amp;section=2" title="Edit section: Perko pair">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable" role="note">Main article: <a href="/wiki/Perko_pair" title="Perko pair">Perko pair</a></div>
<p>In 1974 Perko discovered a duplication in the Tait-Little tables, called the <a href="/wiki/Perko_pair" title="Perko pair">Perko pair</a>.
</p>
<h2><span class="mw-headline" id="New_methods">New methods</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Knot_tabulation&amp;action=edit&amp;section=3" title="Edit section: New methods">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Jim Hoste, <a href="/wiki/Jeffrey_Weeks_(mathematician)" title="Jeffrey Weeks (mathematician)">Jeff Weeks</a>, and <a href="/wiki/Morwen_Thistlethwaite" title="Morwen Thistlethwaite">Morwen Thistlethwaite</a> used computer searches to count all knots with 16 or fewer crossings. This research was performed separately using two different algorithms on different computers, lending support to the correctness of its results. Both counts found 1701936 <a href="/wiki/Prime_knot" title="Prime knot">prime knots</a> (including the <a href="/wiki/Unknot" title="Unknot">unknot</a>) with up to 16 crossings.<sup class="reference" id="cite_ref-htw_1-1"><a href="#cite_note-htw-1">[1]</a></sup>
</p><p>Starting with three crossings (the minimum for any nontrivial knot), the number of prime knots for each number of crossings is
</p>
<dl><dd>1, 1, 2, 3, 7, 21, 49, 165, 552, 2176, 9988, 46972, 253293, 1388705, ... (sequence <span class="nowrap"><a class="extiw" href="//oeis.org/A002863" title="oeis:A002863">A002863</a></span> in the <a href="/wiki/On-Line_Encyclopedia_of_Integer_Sequences" title="On-Line Encyclopedia of Integer Sequences">OEIS</a>)</dd></dl>
