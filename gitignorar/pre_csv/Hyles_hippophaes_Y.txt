<p><i><b>Hyles hippophaes</b></i>, the <b>seathorn hawk-moth</b>, is a species of <a href="/wiki/Moth" title="Moth">moth</a> in the family <a href="/wiki/Sphingidae" title="Sphingidae">Sphingidae</a>. The species was first described by <a href="/wiki/Eugenius_Johann_Christoph_Esper" title="Eugenius Johann Christoph Esper">Eugenius Johann Christoph Esper</a> in 1789. It is found in <a href="/wiki/Afghanistan" title="Afghanistan">Afghanistan</a>, <a href="/wiki/Armenia" title="Armenia">Armenia</a>, <a href="/wiki/Azerbaijan" title="Azerbaijan">Azerbaijan</a>, <a href="/wiki/China" title="China">China</a>, <a href="/wiki/France" title="France">France</a>, <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a>, <a href="/wiki/Germany" title="Germany">Germany</a>, <a href="/wiki/Greece" title="Greece">Greece</a>, <a href="/wiki/Iran" title="Iran">Iran</a>, <a href="/wiki/Iraq" title="Iraq">Iraq</a>, <a href="/wiki/Kazakhstan" title="Kazakhstan">Kazakhstan</a>, <a href="/wiki/Kyrgyzstan" title="Kyrgyzstan">Kyrgyzstan</a>, <a href="/wiki/Mongolia" title="Mongolia">Mongolia</a>, <a href="/wiki/Pakistan" title="Pakistan">Pakistan</a>, <a href="/wiki/Romania" title="Romania">Romania</a>, <a href="/wiki/Serbia_and_Montenegro" title="Serbia and Montenegro">Serbia and Montenegro</a>, <a href="/wiki/Spain" title="Spain">Spain</a>, <a href="/wiki/Switzerland" title="Switzerland">Switzerland</a>, <a href="/wiki/Syria" title="Syria">Syria</a>, <a href="/wiki/Tajikistan" title="Tajikistan">Tajikistan</a>, <a href="/wiki/Turkey" title="Turkey">Turkey</a>, <a href="/wiki/Turkmenistan" title="Turkmenistan">Turkmenistan</a>, and <a href="/wiki/Uzbekistan" title="Uzbekistan">Uzbekistan</a>.
</p><p>The <a href="/wiki/Wingspan" title="Wingspan">wingspan</a> is 65–80 mm. Subspecies <i>H. h. bienerti</i> is paler and browner than related subspecies. A pale, oblique median line is noticeable on the underside of the forewing. The hindwing patches are more orange than red.
</p><p>Larvae of subspecies <i>H. h. bienerti</i> have been recorded on <i><a href="/wiki/Elaeagnus_angustifolia" title="Elaeagnus angustifolia">Elaeagnus angustifolia</a></i> and <i><a href="/wiki/Hippophae_rhamnoides" title="Hippophae rhamnoides">Hippophae rhamnoides</a></i> in China and Tajikistan.
</p>
