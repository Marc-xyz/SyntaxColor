	<h1 class="firstHeading" id="firstHeading" lang="en">WHP Posttranscriptional Response Element</h1>
	
	<div class="mw-body-content" id="bodyContent">
		<div class="noprint" id="siteSub">From Wikipedia, the free encyclopedia</div>
		<div id="contentSub"></div>
		
		
		
		<div id="jump-to-nav"></div>
		<a class="mw-jump-link" href="#mw-head">Jump to navigation</a>
		<a class="mw-jump-link" href="#p-search">Jump to search</a>
		<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><p><b><a href="/wiki/Woodchuck_hepatitis_virus" title="Woodchuck hepatitis virus">Woodchuck Hepatitis Virus</a> (WHP) Posttranscriptional Regulatory Element (WPRE)</b> is a <a class="mw-redirect" href="/wiki/DNA_sequence" title="DNA sequence">DNA sequence</a> that, when transcribed, creates a <a class="mw-redirect" href="/wiki/Tertiary_structure" title="Tertiary structure">tertiary structure</a> enhancing <a href="/wiki/Gene_expression" title="Gene expression">expression</a>. The sequence is commonly used in <a href="/wiki/Molecular_biology" title="Molecular biology">molecular biology</a> to increase expression of <a href="/wiki/Gene" title="Gene">genes</a> delivered by <a href="/wiki/Viral_vector" title="Viral vector">viral vectors</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> WPRE is a tripartite regulatory element with <a href="/wiki/G_beta-gamma_complex" title="G beta-gamma complex">gamma</a>, <a href="/wiki/G_alpha_subunit" title="G alpha subunit">alpha</a>, and beta components.  The alpha component is 80bp long:
</p><p>GCCACGGCGGAACTCATCGCCGCCTGCCTTGCCCGCTGCTGGACAGGGGCTCGGCTGTTGGGCACTGACAATTCCGTGGT<sup class="reference" id="cite_ref-smith_2-0"><a href="#cite_note-smith-2">[2]</a></sup>
</p><p>When used alone without the gamma and beta WPRE components, the alpha component is only 9% as active as the full tripartite WPRE. The sequence for full tripartite WPRE is:
</p><p><small>AATCAACCTCTGGATTACAAAATTTGTGAAAGATTGACTGGTATTCTTAACTATGTTGCTCCTTTTACGCTATGTGGATACGCTGCTTTAATGCCTTTGTA<br/>
TCATGCTATTGCTTCCCGTATGGCTTTCATTTTCTCCTCCTTGTATAAATCCTGGTTGCTGTCTCTTTATGAGGAGTTGTGGCCCGTTGTCAGGCAACGTGGCGTG<br/>
GTGTGCACTGTGTTTGCTGACGCAACCCCCACTGGTTGGGGCATTGCCACCACCTGTCAGCTCCTTTCCGGGACTTTCGCTTTCCCCCTCCCTATTGCCACGGCGG<br/>
AACTCATCGCCGCCTGCCTTGCCCGCTGCTGGACAGGGGCTCGGCTGTTGGGCACTGACAATTCCGTGGTGTTGTCGGGGAAGCTGACGTCCTTTCCATGGCTGCT<br/>
CGCCTGTGTTGCCACCTGGATTCTGCGCGGGACGTCCTTCTGCTACGTCCCTTCGGCCCTCAATCCAGCGGACCTTCCTTCCCGCGGCCTGCTGCCGGCTCTGCGG<br/>
CCTCTTCCGCGTCTTCGCCTTCGCCCTCAGACGAGTCGGATCTCCCTTTGGGCCGCCTCCCCGCCTG</small>
</p><p>This sequence has 100% <a href="/wiki/Sequence_homology" title="Sequence homology">homology</a> with base pairs 1093 to 1684 of the <a class="mw-redirect" href="/wiki/Woodchuck_hepatitis_B_virus" title="Woodchuck hepatitis B virus">Woodchuck hepatitis B virus</a> (WHV8) genome. When used in the 3' untranslated region (UTR) of a mammalian <a href="/wiki/Expression_cassette" title="Expression cassette">expression cassette</a>, it can significantly increase mRNA stability and protein yield.<sup class="reference" id="cite_ref-smith_2-1"><a href="#cite_note-smith-2">[2]</a></sup>
</p>
