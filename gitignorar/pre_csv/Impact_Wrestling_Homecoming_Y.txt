<p><b>Impact Wrestling Homecoming</b> was a <a href="/wiki/Professional_wrestling" title="Professional wrestling">professional wrestling</a> <a href="/wiki/Pay-per-view" title="Pay-per-view">pay-per-view</a> (PPV) <a href="/wiki/List_of_Impact_Wrestling_pay-per-view_events" title="List of Impact Wrestling pay-per-view events">event</a> produced by <a href="/wiki/Impact_Wrestling" title="Impact Wrestling">Impact Wrestling</a>. It took place on January 6, 2019, at <a href="/wiki/Tennessee_State_Fairground_Sports_Arena" title="Tennessee State Fairground Sports Arena">The Asylum</a> in <a href="/wiki/Nashville,_Tennessee" title="Nashville, Tennessee">Nashville, Tennessee</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>
</p>

<h3><span class="mw-headline" id="Background">Background</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Impact_Wrestling_Homecoming&amp;action=edit&amp;section=2" title="Edit section: Background">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Initially Impact Wrestling, then known as Total Nonstop Action Wrestling (TNA), produced weekly pay-per-view shows which operated as the company's main source of revenue, in place of <a href="/wiki/List_of_Impact_Wrestling_pay-per-view_events" title="List of Impact Wrestling pay-per-view events">monthly pay-per-view events</a> used by other promotions. These shows took place mostly at the <a href="/wiki/Tennessee_State_Fairground_Sports_Arena" title="Tennessee State Fairground Sports Arena">Tennessee State Fairground Sports Arena</a> in <a href="/wiki/Nashville,_Tennessee" title="Nashville, Tennessee">Nashville, Tennessee</a>, nicknamed the "TNA Asylum". The shows name comes from their return to the Fairground for the first time since 2010.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>
</p>
<h3><span class="mw-headline" id="Storylines">Storylines</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Impact_Wrestling_Homecoming&amp;action=edit&amp;section=3" title="Edit section: Storylines">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The event featured <a href="/wiki/Professional_wrestling_match_types" title="Professional wrestling match types">professional wrestling matches</a> that involve different wrestlers from pre-existing scripted feuds and <a href="/wiki/Narrative_thread" title="Narrative thread">storylines</a>. Wrestlers portrayed <a href="/wiki/Heel_(professional_wrestling)" title="Heel (professional wrestling)">villains</a>, <a href="/wiki/Face_(professional_wrestling)" title="Face (professional wrestling)">heroes</a>, or <a href="/wiki/Glossary_of_professional_wrestling_terms#Tweener" title="Glossary of professional wrestling terms">less distinguishable characters</a> in the scripted events that built tension and culminated in a wrestling match or series of matches.<sup class="reference" id="cite_ref-HSW_3-0"><a href="#cite_note-HSW-3">[3]</a></sup><sup class="reference" id="cite_ref-Homecoming_4-0"><a href="#cite_note-Homecoming-4">[4]</a></sup>
</p><p>On the November 15, 2018 episode of <i><a href="/wiki/Impact!_(TV_series)" title="Impact! (TV series)">Impact!</a></i>, <a href="/wiki/Brian_Cage" title="Brian Cage">Brian Cage</a> defeated <a href="/wiki/Sami_Callihan" title="Sami Callihan">Sami Callihan</a> to retain the <a href="/wiki/Impact_X_Division_Championship" title="Impact X Division Championship">Impact X Division Championship</a>. After the match, he announced that he would use his Option C privilege, voluntarily vacating the X Division Championship, to challenge <a href="/wiki/John_Morrison_(wrestler)" title="John Morrison (wrestler)">Johnny Impact</a> at Homecoming for the <a href="/wiki/Impact_World_Championship" title="Impact World Championship">Impact World Championship</a>.<sup class="reference" id="cite_ref-optC_5-0"><a href="#cite_note-optC-5">[5]</a></sup>
</p>
