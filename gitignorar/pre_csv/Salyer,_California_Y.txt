<p><b>Salyer</b> is an <a href="/wiki/Unincorporated_area" title="Unincorporated area">unincorporated community</a> in <a href="/wiki/Trinity_County,_California" title="Trinity County, California">Trinity County, California</a>.
</p><p>Salyer is located on <a href="/wiki/California_State_Route_299" title="California State Route 299">State Highway 299</a>, 52 miles (84 km) east of <a href="/wiki/Eureka,_California" title="Eureka, California">Eureka</a> and 93 miles (150 km) west of <a href="/wiki/Redding,_California" title="Redding, California">Redding</a>.
</p><p>Salyer was established on April 16, 1918 and named for Charles Marshall Salyer, a prominent miner.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>
</p><p>The ZIP Code is 95563. The community is inside <a href="/wiki/Area_code_530" title="Area code 530">area code 530</a>.
</p>
<h2><span class="mw-headline" id="Government">Government</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Salyer,_California&amp;action=edit&amp;section=1" title="Edit section: Government">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In the <a href="/wiki/California_State_Legislature" title="California State Legislature">California State Legislature</a>, Salyer is in <a href="/wiki/California%27s_2nd_State_Senate_district" title="California's 2nd State Senate district">the  2nd Senate District</a>, represented by <a href="/wiki/California_Democratic_Party" title="California Democratic Party">Democrat</a> <a href="/wiki/Mike_McGuire_(politician)" title="Mike McGuire (politician)">Mike McGuire</a>, and in <a href="/wiki/California%27s_2nd_State_Assembly_district" title="California's 2nd State Assembly district">the  2nd Assembly District</a>, represented by <a href="/wiki/California_Democratic_Party" title="California Democratic Party">Democrat</a> <a href="/wiki/Jim_Wood_(California_politician)" title="Jim Wood (California politician)">Jim Wood</a>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>
</p><p>In the <a href="/wiki/United_States_House_of_Representatives" title="United States House of Representatives">United States House of Representatives</a>, Salyer is in <a href="/wiki/California%27s_2nd_congressional_district" title="California's 2nd congressional district">California's  2nd congressional district</a>, represented by <a href="/wiki/Democratic_Party_(United_States)" title="Democratic Party (United States)">Democrat</a> <a href="/wiki/Jared_Huffman" title="Jared Huffman">Jared Huffman</a>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>
</p>
<h2><span class="mw-headline" id="Climate">Climate</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Salyer,_California&amp;action=edit&amp;section=2" title="Edit section: Climate">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>This region experiences warm (but not hot) and dry summers, with no average monthly temperatures above 71.6 °F.  According to the <a class="mw-redirect" href="/wiki/K%C3%B6ppen_Climate_Classification" title="Köppen Climate Classification">Köppen Climate Classification</a> system, Salyer has a <a href="/wiki/Mediterranean_climate#Warm-summer_Mediterranean_climate" title="Mediterranean climate">warm-summer Mediterranean climate</a>, abbreviated "Csb" on climate maps.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>
</p>
