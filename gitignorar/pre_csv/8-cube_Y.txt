<p>In <a href="/wiki/Geometry" title="Geometry">geometry</a>, an <b>8-cube</b> is an eight-<a href="/wiki/Dimension" title="Dimension">dimensional</a> <a href="/wiki/Hypercube" title="Hypercube">hypercube</a> (8-cube). It has 256 <a href="/wiki/Vertex_(geometry)" title="Vertex (geometry)">vertices</a>, 1024 <a href="/wiki/Edge_(geometry)" title="Edge (geometry)">edges</a>,  1792 square <a href="/wiki/Face_(geometry)" title="Face (geometry)">faces</a>, 1792 cubic <a class="mw-redirect" href="/wiki/Cell_(mathematics)" title="Cell (mathematics)">cells</a>, 1120 <a href="/wiki/Tesseract" title="Tesseract">tesseract</a> <a class="mw-redirect" href="/wiki/4-face" title="4-face">4-faces</a>, 448 <a href="/wiki/5-cube" title="5-cube">5-cube</a> <a class="mw-redirect" href="/wiki/5-face" title="5-face">5-faces</a>, 112 <a href="/wiki/6-cube" title="6-cube">6-cube</a> <a class="mw-redirect" href="/wiki/6-face" title="6-face">6-faces</a>, and 16 <a href="/wiki/7-cube" title="7-cube">7-cube</a> <a class="mw-redirect" href="/wiki/7-face" title="7-face">7-faces</a>.
</p><p>It is represented by <a href="/wiki/Schl%C3%A4fli_symbol" title="Schläfli symbol">Schläfli symbol</a> {4,3<sup>6</sup>}, being composed of 3 <a href="/wiki/7-cube" title="7-cube">7-cubes</a> around each 6-face. It is called an <b>octeract</b>, a <a href="/wiki/Portmanteau" title="Portmanteau">portmanteau</a> of <a href="/wiki/Tesseract" title="Tesseract">tesseract</a> (the <i>4-cube</i>) and <i>oct</i> for eight (dimensions) in <a href="/wiki/Greek_language" title="Greek language">Greek</a>. It can also be called a regular <b>hexdeca-8-tope</b> or <b>hexadecazetton</b>, being an <a class="mw-redirect" href="/wiki/8-polytope" title="8-polytope">8-dimensional polytope</a> constructed from 16 regular <a href="/wiki/Facet_(geometry)" title="Facet (geometry)">facets</a>.
</p><p>It is a part of an infinite family of polytopes, called <a href="/wiki/Hypercube" title="Hypercube">hypercubes</a>. The <a class="mw-redirect" href="/wiki/Dual_polytope" title="Dual polytope">dual</a> of an 8-cube can be called a <a href="/wiki/8-orthoplex" title="8-orthoplex">8-orthoplex</a>, and is a part of the infinite family of <a href="/wiki/Cross-polytope" title="Cross-polytope">cross-polytopes</a>.
</p>

<p><a class="mw-redirect" href="/wiki/Cartesian_coordinates" title="Cartesian coordinates">Cartesian coordinates</a> for the vertices of an 8-cube centered at the origin and edge length 2 are

<h2><span class="mw-headline" id="Derived_polytopes">Derived polytopes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=8-cube&amp;action=edit&amp;section=4" title="Edit section: Derived polytopes">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Applying an <i><a href="/wiki/Alternation_(geometry)" title="Alternation (geometry)">alternation</a></i> operation, deleting alternating vertices of the octeract, creates another <a href="/wiki/Uniform_polytope" title="Uniform polytope">uniform polytope</a>, called a <i><a href="/wiki/8-demicube" title="8-demicube">8-demicube</a></i>, (part of an infinite family called <a href="/wiki/Demihypercube" title="Demihypercube">demihypercubes</a>), which has 16 <a class="mw-redirect" href="/wiki/Demihepteract" title="Demihepteract">demihepteractic</a> and 128 8-simplex facets.
</p>
