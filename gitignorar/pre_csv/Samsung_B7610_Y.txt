<p><b>Samsung B7610</b> (also known as <b>Samsung Omnia Pro B7610</b>) is a <a href="/wiki/Smartphone" title="Smartphone">smartphone</a> produced by <a href="/wiki/Samsung" title="Samsung">Samsung</a> as part of their Omnia Series line of mobile phones. The B7610 Omnia<sup>PRO</sup> runs <a href="/wiki/Windows_Mobile" title="Windows Mobile">Windows Mobile</a> 6.1 or 6.5 Professional, with a TouchWiz 2.0 <a href="/wiki/User_interface" title="User interface">user interface</a>. The phone has a physical and virtual <a href="/wiki/QWERTY" title="QWERTY">QWERTY</a> keyboard.
</p><p>Its capabilities include:<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> a <a href="/wiki/Global_Positioning_System" title="Global Positioning System">Global Positioning System</a> receiver with maps and optional <a href="/wiki/Turn-by-turn_navigation" title="Turn-by-turn navigation">turn-by-turn navigation</a>; an autofocus 5-<a class="mw-redirect" href="/wiki/Megapixel" title="Megapixel">megapixel</a> <a href="/wiki/Digital_camera" title="Digital camera">digital camera</a> with LED flash, <a href="/wiki/Flash_(photography)" title="Flash (photography)">flash</a>, video recording and <a class="mw-redirect" href="/wiki/Video_conferencing" title="Video conferencing">video conferencing</a>; wireless connectivity via <a class="mw-redirect" href="/wiki/HSDPA" title="HSDPA">HSDPA</a>, <a class="mw-redirect" href="/wiki/DLNA" title="DLNA">DLNA</a>, Wi-Fi 802.11 b/g and <a href="/wiki/Bluetooth" title="Bluetooth">Bluetooth</a>; a <a href="/wiki/Portable_media_player" title="Portable media player">portable media player</a> with the ability to download files over the air; <a class="mw-redirect" href="/wiki/Composite_Video" title="Composite Video">Composite Video</a> output via optional cable; <a href="/wiki/Computer_multitasking" title="Computer multitasking">multi-tasking</a> to allow several applications to run simultaneously; a <a href="/wiki/Web_browser" title="Web browser">web browser</a> with support for <a href="/wiki/HTML" title="HTML">HTML</a>, <a href="/wiki/JavaScript" title="JavaScript">JavaScript</a> and <a href="/wiki/Adobe_Flash" title="Adobe Flash">Adobe Flash</a>; messaging via <a href="/wiki/SMS" title="SMS">SMS</a>, <a href="/wiki/Multimedia_Messaging_Service" title="Multimedia Messaging Service">MMS</a> and <a class="mw-redirect" href="/wiki/E-mail" title="E-mail">e-mail</a>; <a class="mw-redirect" href="/wiki/Office_suite" title="Office suite">Office suite</a> and <a class="mw-redirect" href="/wiki/Electronic_Organizer" title="Electronic Organizer">organizer</a> functions; and the ability to install and run third party <a class="mw-redirect" href="/wiki/Java_ME" title="Java ME">Java ME</a> or Windows Mobile applications.
</p>
